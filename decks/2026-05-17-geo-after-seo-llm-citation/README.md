# 後 SEO 時代：當品牌要被 LLM 引用、不只被 Google 排名

> After SEO: The LLM Citation Economy and Generative Engine Optimization — 2026-05-17

## Hook

搜尋從藍色連結遷移到 AI Overviews / ChatGPT / Perplexity。71% 美國人用 AI 評估品牌、AI Overviews 出現在 ~13% 美國桌面搜尋。對 SEO 背景的人、2026 iSchool 的入場券是 GEO——LLM citation behavior、retrieval-augmented generation、attribution when answers replace links。這是 JASIST 跟 iSchool faculty 真的在寫的東西。

## Theoretical anchor

Aggarwal et al. 'GEO: Generative Engine Optimization' (Princeton, arXiv 2311.09735, 2024-2025 follow-ups); NBER w34255 'How People Use ChatGPT' (Chatterji, Cunningham, Deming et al., Sept 2025); 2025 JASIST AI-mediated info seeking 系列。經典 Wilson/Kuhlthau information-behavior 模型被點名 break — 因為使用者不再 formulate keyword query。

## Implications

- **HCI:** AI Overview vs SERP 的 UI affordance diff、citation surface design、user trust calibration — 直接是 CHI / CSCW 題目
- **CS:** IR retrieval + ranking explanation + LLM-as-search 的 benchmark 缺 — JASIST 跟 CS IR community 可以共建 dataset
- **Critical Data Studies:** platform 把 '誰被引用' 從 PageRank 演算法外包到 LLM 訓練分布 — epistemic visibility 的 platform capture

- **Marketing / luxury angle:** GEO 是品牌 discovery 新戰場；對 luxury 是 paywalled / off-platform content 怎麼還被 ChatGPT 引用

## Key findings

- GEO formal definition (Princeton 2023, expanded 2024-25)：optimize to be INSIDE the answer, not above it
- Information-seeking 佔 ChatGPT messages 從 14% → 24% 一年內 (NBER w34255 2025)
- AI visitor 比 search visitor 高 4.4× conversion (Salesforce State of Marketing 2026)
- E-E-A-T 從 Google 遷移到 LLM — author bios + citations 仍是最強 GEO signal
- Mid-market GEO 預算 $75k-$150k/year、enterprise $250k+ (Walker Sands 2025)
- 51% 消費者說 GenAI 改了他們研究習慣、71% 改了 query phrasing (Search Engine Land 2025)
- iConference 2025 'Living in an AI-gorithmic world' (Indiana Bloomington) 把 GEO 跟 LIS information behavior 接起來
