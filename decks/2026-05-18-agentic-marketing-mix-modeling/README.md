## Implications

- **HCI:** Marketing dashboard 讓位給 natural-language console 給 non-technical CMO interrogate model — 十年 interface design 問題
- **CS:** 生產 MMM stack 現在要 explainability tool (SHAP) + agent orchestration (Salesforce/Sellforte) + simulation environment — CS-trained marketer 的三軸能力
- **Critical Data Studies:** Pasteur 象限 CDS thesis：在同一個 retail dataset 上比 SHAP-attribution vs Bayesian causal MMM — 可發、可實用

## Key findings

- AI-enhanced MMM (Feb 2026 preprint)：R² 從 0.857 → 0.912、RMSE 降 21.7%、MAE 降 22.8% vs OLS、報告 18-25% marketing ROI 提升 with SHAP+LLM integration
- 53.5% 美國 marketer 已用 MMM 處理 post-cookie privacy limit (PracticalEcommerce 2025) — MMM 是 replacement attribution stack、不是 niche tool
- Sellforte (Oct 2025) 上 specialized agents：Media Planner / Media Buyer / Experiments — 執行從 analyst 轉到 agent
- Salesforce Agentforce 360 (Oct 13, 2025) introduce Marketing Cloud Next agentic agents『build and launch campaigns, journeys, assets autonomously』
- arXiv 2510.18155 (Oct 2025) demo LLM-based multi-agent simulation of consumer behavior — pre-implementation testing replace high-risk a/b test
- Hershey 公開 commit agentic AI optimize $2B marketing spend (Adweek 2025) — Fortune-100 scale production validation
- Luxury 角度：first-party data 薄的品牌不成比例受益 MMM 4.0 — 不需要 user-level tracking
