## Implications

- **HCI:** agent UX storyboard、preference elicitation、multi-turn dialog evaluation — CHI / CSCW 直接題目
- **CS:** agent tool use benchmark 缺 luxury commerce scenario — LIS / business school 可貢獻
- **Critical Data Studies:** 'agent 替你買' 把 consumer agency 外包 — 行為資料權與 platform 治理交叉

## Key findings

- Shopping search on GenAI 漲 4,700% YoY (BSPK 2026 trend report)
- Agentic commerce in fashion：3× conversion、38% AOV uplift (Alhena 2025 deployments)
- Kering KNXT + Madeline 跨 Kering portfolio (2025)
- Alta 'fragrance wardrobe' stylist agent — personal context > seasonal narrative
- Agentic AI market：$7.29B、40.5% CAGR (Gartner / 多家 2025 synthesis)
- 75% marketer 用 AI、但 84% 仍跑 generic campaign — agent layer 補這個 gap (Salesforce 2026)
- Open governance question：當 agent 替你買、customer of record 是誰？
