## Hook

Luxury resale 2024 市場 $32.47B、預計 2030 漲到 $50B——但這只有在『你信你買到的東西』成立才能運作。讓這信任成立的基礎建設是 AI-driven 微距鑑定（Entrupy）、150 個專家 hybrid human+AI hub（Vestiaire Collective）、跟 Aura DPP 收斂。這是 2025-2026 最可量測的 AI-luxury 交叉點、而且大多 HCI/CS 讀者沒看到。

## Implications

- **HCI:** Resale UX 需要 progressive disclosure of authentication evidence — 買家要看 microscope image、不只是 green checkmark
- **CS:** Microscopic-imaging CNN + blockchain DPP 在生產形成 deployed adversarial ML — counterfeiter 訓練對抗
- **Critical Data Studies:** CDS 角度：authentication 經濟可 end-to-end 量測（item → DPP → resale → return）— luxury domain 罕見的 clean longitudinal data

## Key findings

- Luxury resale：$32.47B (2024) → $50.06B (2030)、7.48% CAGR — AI authentication 是承重基礎建設 (ResearchAndMarkets 2025-2030 report)
- Entrupy 用 microscopic imaging + CNN 為 20+ top luxury 品牌做 scale 鑑定；當 API / device 賣給商家 (2025 industry data)
- Vestiaire Collective 在 5 個城市（US/UK/France/HK/Singapore）跑 150-expert authentication hub — hybrid human + AI 是 trust default、不是 full automation
- Aura Blockchain Consortium 2025 達 60M+ 商品、50+ 會員品牌 — DPP 從 marketing add-on 變成 live by default
- TikTok Shop 2024-2025 進 luxury resale、leveraging 外部 authenticator — trust stack 從 marketplace unbundling (RetailBrew / EcommOp 2024-2025)
- EU Ecodesign / DPP mandate 2026-2027 phase-in — 過去的 luxury differentiator 變成 EU-wide compliance、AI authentication 變 operational bottleneck
- Counterfeiter 反過來訓練模型對抗 Entrupy 分類器 — 一個 deployed adversarial ML system in production
